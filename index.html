About Me

Hi! I'm Carlos, an undergrad in UP Diliman, majoring in computer science.

I have a passion for teaching, learning, and all things academic. I believe in working smart instead of working hard, in one way by applying deeper CS theory to optimize the stuff I make.

My software development journey started in high school by making our school's computerized student election system, and I've been making different kinds of software for the community ever since.

Outside of that, I enjoy scuba diving and playing my violin!

My Solo Projects

UP DMMME Request Form System 2020

A Laravel 7 web application that organizes the request form lifecycle for the laboratories in UP's Department of Mining, Metallurgical, and Materials Engineering. It features secure request tracking, automated emailing, unit testing, user action logging, document generation, among many others. The repository is currently private, and deployment is scheduled in late 2020 or early 2021.

Shalltear 2020

A Discord bot for a farming game with stock market-esque mechanics, all made in discord.py. Showcases algorithm design, interfacing with Discord using the discord.py framework, and graph generation using Python 3 libraries.

Kyu 2018

A queueing system made using socket.io, Node.js, and Electron. Deployed for a client in 2018, the kiosk screen that shows the queue serves as the central units, with the client units queueing up numbers through the local area network.
